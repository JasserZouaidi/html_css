HTML5

HyperText Markup Language 5
est la derniere version majeure du HTML (format de données couçu pour représenter les pages web ).
Cette version a été finalisée le 28 octobre 2014.

Evolution du language :

  1. 1989-1992 : origine
  2. 1993 : HTML 1.0
  3. 1995-1996 : HTML 2.0
  4. 1997 : HTML 3.2 et 4.0
  5. 2000 : XHTML
  6. De 2007 a nos jours :
    1. HTML 5
    2. Abondon du XHTML 2
